# Heart Disease Prediction

## Overview
Predict whether a person is at risk of heart disease using ML models.

## Features
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Models:
  - Logistic Regression
  - Decision Tree
- Evaluation:
  - Accuracy
  - Confusion Matrix
  - ROC-AUC

## Tech Stack
Python, Pandas, Scikit-learn, Seaborn, Matplotlib

## Results
Logistic Regression achieved higher ROC-AUC and better generalization.

## Future Improvements
- Use Random Forest / XGBoost
- Hyperparameter tuning
- Deploy as web app



# Stock Price Prediction

## Overview
Predict next-day stock closing prices using machine learning models.

## Features
- Data fetched using yfinance
- Feature engineering (OHLC + Volume)
- Models:
  - Linear Regression
  - Random Forest
- Evaluation using RMSE
- Visualization of predictions

## Tech Stack
Python, Pandas, Scikit-learn, Matplotlib, yfinance

## Results
Random Forest performed better with lower RMSE.

## Future Improvements
- Add LSTM (Deep Learning)
- Include technical indicators (RSI, MACD)
